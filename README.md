# wifi-go-nmcli
compact go program to connect wifi with nmcli for unix-like env

### Install
1. golang

**Arch Linux** <br>

```
sudo pacman -S go
git clone https://github.com/akil4n/wifi-go-nmcli.git
cd wifi-go-nmcli
go build wifi.go
sudo mv wifi /usr/bin/
```


**Ubuntu Linux 20.04**<br>

```
sudo apt install golang-go
git clone https://github.com/akil4n/wifi-go-nmcli.git
cd wifi-go-nmcli
go build wifi.go
sudo mv wifi /usr/bin/
```
